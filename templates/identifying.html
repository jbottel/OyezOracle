Learning From The Transcript

Any statements that are made by the justices are telling. They sometimes will cut off a lawyer while they are speaking, although sometimes they will wait for a pause. The justices each have their own personality. Justice Clarence Thomas is well-known for not speaking at all during oral argument. Sometimes a justice will ask "why" or "how", indiciating that they want the advocate to further explain. Sometimes a justice will say words like "wrong," which could be an indiciator that they disagree with what the advocate is saying.

It is features like these that OyezOracle uses to make its predictions. A listing of features can found below:

  • Number of statements per argument
  • Number of words per statement
  • Number of times a statement was an interruption
  • Number of occurences of the phrase "Don't Know"
  • Number of occurences of the word "Why"
  • Number of occurences of the word "Wrong"
  • Number of nouns, proper nouns, etc.
  • Number of verbs, adjectives, and adverbs.

Technical Details

Natural Language Processing

OyezOracle uses the nltk package to perform natural language processing. The major extent of this process is to identify the number of times a specific part-of-speech is used. In fact, this is the most processor-intensive part of the transcript analysis.

Features from NLTK

OyezOracle uses a set of feature codes to build a vector of features as a neural network input. A significant portion of these features is made up of the NLTK-determined parts-of-speech. The NLTK feature codes can be seen below.

Martin-Quinn Scores

An additional feature considered by OyezOracle is called a Martin-Quinn score. This is a score, created in process described by Martin and Quinn, which gives a numerical measure of the liberal/conservative ideology of a justice. These scores allow for some measure of ideological concerns to come into the prediction, which is a very obvious indicator to anyone who is familiar with the supreme court. 

mq_scores = {
    "ALTO": 2.037,
    "BRY": -1.729,
    "GIN": -2.461,
    "KAG": -1.913,
    "KEN": -0.150,
    "ROBT": 0.869,
    "SCAL": 1.583,
    "SOTO": -2.354,
    "THOM": 3.240,
}
